# docs-debug

#### 介绍

本仓库提供了 openeuler 的文档捉虫代码，可用于文档的捉虫功能的展示。

#### 目录结构

website
├─ css // 页面的 css 文件
├─ img // 页面的图片文件
├─ js // 页面的 js 文件
├─ index.html // 页面

#### 使用说明

1.  clone 仓库代码
2.  找到 website 路径下的 index.html
3.  双击找到的 index.html 即可在浏览器中打开，查看展示效果
